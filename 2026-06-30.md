### PhysisForcing: Physics Reinforced World Simulator for Robotic Manipulation
- 类型：论文
- 方向：多模态
- 发布日期：2026-06-26
- 来源与作者：arXiv；Peiwen Zhang, Yufan Deng, Shangkun Sun 等
- 一句话结论：用物理一致性监督强化视频生成模型，可显著提升机器人操作世界模拟与闭环规划效果。
- 核心内容：论文指出通用视频生成器和机器人微调模型仍会产生轨迹不连续、接触关系不可信等物理错误。PhysisForcing 通过像素级轨迹对齐损失和语义级关系对齐损失，让 DiT 特征更关注物理信息区域。作者报告在 R-Bench、PAI-Bench、EZS-Bench 上均提升生成质量，并在 WorldArena action-planner 协议下把闭环成功率从 16.0% 提升到 24.0%。
- 创新点：物理信息区域监督；轨迹与关系双层对齐；把视频世界模型接入机器人规划评测。
- 为什么值得关注：它把视频生成从“看起来合理”推进到“可辅助具身决策”，是多模态世界模型落地机器人的关键方向。
- 局限与疑点：闭环成功率仍不高；主要依赖模拟/基准环境，真实机器人泛化待验证。
- 开源情况：论文标注 GitHub 与项目页已提供。
- 原始链接：https://arxiv.org/abs/2606.28128

### ProMSA: Progressive Multimodal Search Agents for Knowledge-Based Visual Question Answering
- 类型：论文
- 方向：两者相关
- 发布日期：2026-06-26
- 来源与作者：arXiv；ZhengXian Wu, Hangrui Xu, Kai Shi 等
- 一句话结论：KB-VQA 不应固定 top-k 检索，模型可学习何时图搜、文搜或停止。
- 核心内容：ProMSA 把知识型视觉问答建模为渐进式多模态搜索 Agent。给定图像和问题，Agent 在显式工具预算下迭代选择 image search、text search 或 stop，并通过去重减少冗余检索。训练上先用 rejection-sampling SFT 学会工具格式，再用 TN-GSPO 做序列级 RL。作者称在 E-VQA 与 InfoSeek 上超过强 RAG 和 Agent 基线。
- 创新点：自适应多模态检索策略；工具深度归一化 RL；显式预算与去重机制。
- 为什么值得关注：它把 RAG 从静态管线推进到可控工具调用，对多模态 Agent 的检索规划很有参考价值。
- 局限与疑点：收益是否来自更强检索覆盖还是推理策略本身，需要更多消融；开放域搜索成本也需评估。
- 开源情况：论文称代码已开放。
- 原始链接：https://arxiv.org/abs/2606.27974

### PerceptionRubrics: Calibrating Multimodal Evaluation to Human Perception
- 类型：论文
- 方向：多模态
- 发布日期：2026-06-26
- 来源与作者：arXiv；Yana Wei, Hongbo Peng, Yanlin Lai 等；ICML 2026
- 一句话结论：多模态评测应从整体语义相似转向原子事实审计，否则高分会掩盖感知脆弱性。
- 核心内容：PerceptionRubrics 用 1,038 张高信息密度图像和 12,000 多条实例级 rubrics 评测视觉理解。标准由 Circular Peer-Review 共识流程生成，并分成 Must-Right 与 Easy-Wrong 两类。Gated Scoring 对关键视觉事实失败施加强惩罚。作者发现模型能验证碎片元素，却常在严格合取约束上失败；开放模型与闭源前沿仍有约 8% 感知差距。
- 创新点：实例级原子视觉事实 rubrics；Must-Right/Easy-Wrong 双流；门控评分更贴近人类感知。
- 为什么值得关注：它直接挑战“榜单饱和”的错觉，适合作为 VLM 与图像生成评测的新补充。
- 局限与疑点：数据规模不大；rubric 构造成本高，领域迁移仍需观察。
- 开源情况：论文提供项目页，未确认完整数据/代码开放程度。
- 原始链接：https://arxiv.org/abs/2606.28322

### Reflect-R1: Evidence-Driven Reflection for Self-Correction in Long Video Understanding
- 类型：论文
- 方向：多模态
- 发布日期：2026-06-26
- 来源与作者：arXiv；Shuimu Chen, Yuteng Chen, Yuanshen Guan 等；ECCV
- 一句话结论：长视频理解的自我纠错需要外部视觉证据，而不是只靠模型内部反思。
- 核心内容：Reflect-R1 构建 intuition、verification、arbitration 三阶段流程，通过动态检索客观视觉证据校验初始直觉，并执行多次时间搜索解决冲突。为降低多阶段策略耦合，作者提出 SD-GRPO，分别计算不同推理阶段的优势函数。论文还构建 120K 样本训练数据，并报告在 VideoMME、LongVideoBench 等基准上达到 SOTA。
- 创新点：证据驱动长视频自纠错；阶段解耦 RL；面向反思训练的数据构建。
- 为什么值得关注：视频 Agent/多模态助手常因长上下文幻觉失败，这类“证据检索+反思”框架更接近可审计推理。
- 局限与疑点：外部证据检索质量会成为瓶颈；计算成本和实时性未必适合在线应用。
- 开源情况：arXiv 页面未显示代码链接。
- 原始链接：https://arxiv.org/abs/2606.27922

### AirGroundBench: Probing Spatial Intelligence in Multimodal Large Models under Heterogeneous Multi-View Embodied Collaboration
- 类型：论文
- 方向：多模态
- 发布日期：2026-06-26
- 来源与作者：arXiv；Haotian Li, Yida Wang, Leyuan Wang 等
- 一句话结论：当前 MLLM 在空地多视角协作中仍缺乏几何一致的空间理解。
- 核心内容：AirGroundBench 面向 UAV-UGV 协作，包含 11 个高保真模拟环境、1,021 组同步空地观察、约 62,000 个双视角 VQA 实例和 115 个闭环导航 episode。任务覆盖空间感知、跨视角对齐、空间变换推理和具身决策。13 个 MLLM 的评测显示，模型在基础空间感知上尚可，但跨视角对齐和变换推理明显薄弱，并会传导到导航决策。
- 创新点：异构空地双视角具身评测；结构化 2D/3D 空间标注；把 VQA 与闭环导航结合。
- 为什么值得关注：多智能体具身系统需要共享空间表征，这个基准能暴露单视角 VLM 看不到的问题。
- 局限与疑点：主要来自模拟环境；真实传感器噪声、通信延迟和控制闭环复杂度仍未充分覆盖。
- 开源情况：arXiv 页面未显示明确代码链接。
- 原始链接：https://arxiv.org/abs/2606.28049

### ATOD: Annealed Turn-aware On-policy Distillation for Multi-turn Autonomous Agents
- 类型：论文
- 方向：Agent
- 发布日期：2026-06-26
- 来源与作者：arXiv；Qitai Tan, Zefang Zong, Yang Li, Peng Chen
- 一句话结论：小模型 Agent 训练可用“先模仿、后强化”的退火策略兼顾学习效率与最终上限。
- 核心内容：ATOD 针对长程交互任务中 OPD 早期快但上限低、RL 上限高但反馈稀疏的问题，提出 annealed OPD-RL schedule。早期由 OPD 主导以接近教师行为，后期逐步增强 RL 以探索更高奖励。T-DUR 机制会放大长轨迹中高价值 turn 的监督权重。作者在 ALFWorld、WebShop、Search-QA 上报告平均成功率超过 OPD 3.03 点、超过 GRPO 23.62 点。
- 创新点：退火式 OPD-RL 混合训练；turn 级分歧不确定性重加权；面向多轮 Agent 的小模型后训练。
- 为什么值得关注：如果结果稳定，它为低成本 Agent 部署提供了比纯 RL 更实际的训练路径。
- 局限与疑点：依赖教师质量；是否能迁移到浏览器/桌面 Computer Use 还需验证。
- 开源情况：arXiv 页面未显示代码链接。
- 原始链接：https://arxiv.org/abs/2606.27814
