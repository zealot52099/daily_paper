### TOPS: First-Principles Visual Token Pruning via Constructing Token Optimal Preservation Sets for Efficient MLLM Inference
- 类型：论文
- 方向：多模态
- 发布日期：2026-06-25
- 来源与作者：arXiv；Tinghao Wang、Yichen Guo、Rui Huang、Zheng Lu、Qizhe Zhang 等，Peking University / UESTC / NTU / BAAI
- 一句话结论：TOPS 用训练无关、模型无关的视觉 token 子集选择，在大幅压缩 MLLM 视觉 token 时尽量保持甚至改善性能。
- 核心内容：论文把视觉 token pruning 重新表述为 Token Optimal Preservation Sets 构造问题，提出 Task Relevance、Information Coverage、Semantic Diversity 三个原则。实验覆盖 7 个 MLLM backbone 和 14 个 benchmark。作者报告在 LLaVA-NeXT 上移除 77.8% 视觉 token 后，7B/13B 性能保持 100.0%/100.6%。
- 创新点：信息论视角定义剪枝目标；两阶段 text-aware token 选择；训练无关、可插拔。
- 为什么值得关注：MLLM 推理成本仍是落地瓶颈，这类方法可能直接影响端侧、多图和视频理解部署。
- 局限与疑点：需要进一步验证在闭源强模型、超长视频和真实服务延迟上的收益。
- 开源情况：arXiv 页面未见明确代码链接。
- 原始链接：https://arxiv.org/abs/2606.27161

### AgentX: Towards Agent-Driven Self-Iteration of Industrial Recommender Systems

- 方向：Agent
- 发布日期：2026-06-25
- 来源与作者：arXiv；Changxin Lao、Fei Pan、Guozhuang Ma、Han Li、Kun Gai、Mo Zhou、Quan Chen 等 60 位作者
- 一句话结论：AgentX 把推荐系统实验从人工工程循环推进到多 Agent 自迭代闭环。
- 核心内容：系统包含 Brainstorm、Developing、Evaluation 与 Harness Evolution 四阶段。Agent 自动提出实验、修改生产代码、做可靠性验证、发起带 guardrail 的 A/B 评估，并将成败经验沉淀为结构化知识。论文强调该系统已面向工业推荐场景部署。
- 创新点：把 Agent 用于真实推荐算法生产闭环；将在线 A/B 结果转化为知识资产；SGPO 蒸馏执行轨迹以改进 Agent。
- 为什么值得关注：这是 Agent 从 demo 走向企业核心研发流程的典型案例。
- 局限与疑点：生产系统细节、安全边界、失败率和人工介入比例仍需更多可复现实证。
- 开源情况：未见代码开放。

### Joint Learning of Experiential Rules and Policies for Large Language Model Agents
- 类型：论文
- 方向：Agent
- 发布日期：2026-06-25
- 来源与作者：arXiv；Shicheng Ye、Chao Yu
- 一句话结论：JERP 同时更新经验规则池和 Agent policy，缓解“外部规则”和“参数学习”脱节。
- 核心内容：论文关注多步交互环境中如何使用历史轨迹。JERP 在决策时检索任务相关规则，并在 episode 后用当前轨迹与成功参考轨迹比较来修订规则池，同时优化 policy。AlfWorld 与 WebShop 实验显示复杂交互任务决策表现有稳定提升。
- 创新点：规则池与策略联合学习；用成功轨迹对比更新经验；兼顾可解释规则与参数吸收。
- 为什么值得关注：Agent 长期学习需要比 RAG 记忆更稳定的经验积累机制。
- 局限与疑点：实验环境仍偏标准 benchmark，真实网页和工具生态迁移性待验证。
- 开源情况：未见代码开放。
- 原始链接：https://arxiv.org/abs/2606.27136

### Memory Depth, Not Memory Access: Selective Parametric Consolidation for Long-Running Language Agents
- 类型：论文
- 方向：Agent
- 发布日期：2026-06-25
- 来源与作者：arXiv；Haoliang Han
- 一句话结论：作者区分“记忆访问”和“记忆深度”，主张长期 Agent 需要把关键经验选择性写入小型参数存储。
- 核心内容：论文提出 loop-drift protocol：检索索引保持完整，但工作上下文卸载后仍要维持目标行为。EVAF 使用 surprise 与 valence gate 做 LoRA consolidation。作者报告检索更擅长浅层事实回忆，而 EVAF 在 goal persistence 与 post-unload recovery 上更强。
- 创新点：提出 memory depth 评测视角；稀疏参数写入；区分 selection 与 actuation 两个机制维度。
- 为什么值得关注：长期 Agent 记忆不能只靠向量库，参数化巩固可能成为下一类记忆层。
- 局限与疑点：主要基于小模型与受控协议，陈旧记忆失效仍是未解决边界。
- 开源情况：未见代码开放；附 supplementary PDF。
- 原始链接：https://arxiv.org/abs/2606.26806

### Semantic Early-Stopping for Iterative LLM Agent Loops
- 类型：论文 / 开源项目
- 方向：Agent
- 发布日期：2026-06-25
- 来源与作者：arXiv；Sahil Shrivastava
- 一句话结论：语义早停可减少多 Agent 循环 token 消耗，但“何时停”之外，“哪一轮最好”仍是开放问题。
- 核心内容：论文用连续 draft embedding 的 cosine distance 与质量变化判断 Writer-Critic 循环是否收敛。HotpotQA 60 题测试中，judge-free semantic stopper 相比固定 max_iterations 节省 38% operational tokens，质量基本持平；带质量 gate 的版本因逐轮 judge 成本反而不划算。
- 创新点：语义收敛早停；trajectory replay 与 judge cache 的低成本评测；机器检查终止性与 well-definedness。
- 为什么值得关注：多 Agent 系统成本很大一部分来自无效迭代，早停策略有直接工程价值。
- 局限与疑点：样本规模较小，评测依赖 LLM/RAGAS judge，尚未覆盖代码、浏览器操作等复杂 Agent。
- 开源情况：已开放实现：https://github.com/SahilShrivastava-Dev/semantic-halting-problem
- 原始链接：https://arxiv.org/abs/2606.27009

### EO-WM: A Physically Informed World Model for Probabilistic Earth Observation Forecasting
- 类型：论文
- 方向：多模态
- 发布日期：2026-06-25
- 来源与作者：arXiv；Junwei Luo、Shuai Yuan、Zhenya Yang、Yansheng Li、Zhe Liu、Hengshuang Zhao
- 一句话结论：EO-WM 将卫星 Earth Observation 预测建模为受天气驱动的概率世界模型问题。
- 核心内容：模型采用 video diffusion transformer 进行多光谱 EO 预测，把气象 forcing 分解为 climatological baseline、weather anomalies 和累计物理 stress。论文新增 Extreme Summer 与 Seasonal Matched-Pair 两个诊断 benchmark，评估模型是否对天气变化作出正确响应。作者报告 NDVI decline amplitude error 相对降低 5.63%，directional hit rate 相对提升 7.80%。
- 创新点：物理先验条件分解；面向天气响应的诊断评测；多光谱视频扩散世界模型。
- 为什么值得关注：世界模型正在从通用视频生成扩展到科学预测与环境监测。
- 局限与疑点：地理泛化、极端事件数据稀缺和长期预测稳定性仍需检验。
- 开源情况：论文称 benchmarks 与模型将开源，arXiv 页面提供 GitHub 链接但本次未能可靠解析。
- 原始链接：https://arxiv.org/abs/2606.27277

### Einstein World Models
- 类型：论文
- 方向：两者相关
- 发布日期：2026-06-25
- 来源与作者：arXiv；Munachiso Samuel Nwadike、Zangir Iklassov、Ali Mekky、Zayd M. Kawakibi Zuhri、Kentaro Inui
- 一句话结论：EWM 提出让 LLM 调用视觉-时间 rollout 模块，把“视觉思想实验”纳入推理链。
- 核心内容：论文认为部分复杂推理不能只靠语言充分表达。Einstein World Models 不是直接输出答案，而是让 LLM 调用 world-module 生成短场景 rollout，把它作为可检查假设继续推理。作者将其类比为从 web search/code execution 扩展到 visual thought experiments 的工具调用。
- 创新点：把视觉 rollout 放入 reasoning trace；将世界模块视为推理工具；强调反事实可视化。
- 为什么值得关注：这与多模态 CoT、可交互世界模型和 Agent tool use 三条线汇合。
- 局限与疑点：目前更像蓝图，缺少大规模实验和可复现系统验证。
- 开源情况：未见代码开放。
- 原始链接：https://arxiv.org/abs/2606.26969
