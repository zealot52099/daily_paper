### H2HMem: A Multimodal Memory Benchmark for Agents in Human-Human Interactions
- 类型：论文
- 方向：两者相关
- 发布日期：2026-06-08
- 来源与作者：arXiv；Shiping Zhu、Yibo Yang、Zhengyang Wang、Tiancheng Shen、Dandan Guo、Ming-Hsuan Yang
- 一句话结论：现有 Agent memory 在跨模态、跨参与者、跨会话的信息组织与推理上仍有明显缺口。
- 核心内容：H2HMem 面向“Agent 旁观人类之间交互”的场景，包含双人及多人、多会话、文本与图像混合对话。数据含 25 组对话、309 个会话、7,078 轮对话和 2,236 个 QA，评测覆盖 recall、reasoning、application 三类九项能力。作者报告最优方法的总体 LLM-Judge 分数仅 0.5757；外部记忆能提高召回，但跨模态检索、冲突检测和结构化推理仍弱。
- 创新点：首次系统结合多人交互、长程会话和多模态记忆；将记忆评测从事实召回扩展到知识演化、因果推理、冲突检测与拒答。
- 为什么值得关注：它更接近会议助手、临床记录等真实 Agent 场景，可能成为长期多模态记忆的重要压力测试。
- 局限与疑点：主要对话由 LLM 生成并经人工校验，真实交流分布、隐私噪声和口语音频尚未充分覆盖；多人部分仅 5 组对话。
- 开源情况：论文声明开放代码、数据集与榜单，数据采用 CC BY；具体入口见论文页面。
- 原始链接：https://arxiv.org/abs/2606.09461

### Scaling by Diversified Experience for Vision-Language-Action Models
- 类型：论文
- 方向：多模态
- 发布日期：2026-06-08
- 来源与作者：arXiv；Leiyu Wang、Zhaofengnian Wang、Xueqi Li、Luoyi Fan、Cewu Lu、Nanyang Ye
- 一句话结论：SyVLA 试图同时解决 VLA 中“推理干扰控制”和真实机器人 RL 易崩溃两类问题。
- 核心内容：模型以 Qwen2.5-VL-3B 加 Flow Matching Action Expert 构成双系统，通过 Feature Query Tokens 传递控制意图。Intention Decoupling 根据 action loss 对 token hidden state 的梯度范数屏蔽低相关表示；Similar-Sample Guided RL 从 imitation learning 数据检索相似样本约束更新。作者报告长程 Folding-Shirt 任务相对 IL 初始化最高提升 15 个百分点，并在该任务达到 0.86 成功率。
- 创新点：用梯度信号显式分离高层推理与低层控制；用相似示范样本稳定大规模 VLA 的在线 RL；兼顾动作能力与通用视觉语言能力保留。
- 为什么值得关注：VLA 的关键瓶颈正从“能否生成动作”转向“推理、控制与在线优化能否共存”，该工作给出可操作的训练方案。
- 局限与疑点：真实机器人任务规模有限，部分强基线结果依赖作者复现；Intention Decoupling 在通用多模态基准上的增益并不一致，理论分析也基于简化 Action Expert。
- 开源情况：项目页已上线；作者承诺开放代码和预训练数据。https://sy-vla.github.io/
- 原始链接：https://arxiv.org/abs/2606.09009

### SpaceVLN: A Zero-Shot Vision-and-Language Navigation Agent with Online Spatial Cognitive Memory and Reasoning
- 类型：论文
- 方向：两者相关
- 发布日期：2026-06-08
- 来源与作者：arXiv；Yucheng Deng、Pingrui Lai、Xinhai Li、Chenjia Bai、Xiaoheng Deng、Chengnuo Sun、Xuelong Li、Hua Yang
- 一句话结论：把导航历史组织成在线空间认知记忆，比单纯堆叠视觉历史更适合零样本具身导航。
- 核心内容：SpaceVLN 以可验证的 space-landmark stage 分离规划和执行，并把已探索区域抽象成 Spatial Waypoints、路径和局部 landmark memory。Spatial-CoT 联合任务进度、空间感知和下一阶段预测，实现闭环重规划。作者报告在 R2R-CE、RxR-CE、GN-Bench、HM3D-OVON 上取得零样本 SOTA；真实机器人 50 个 episode 的总体成功率为 48%，高于所列基线的 34%。
- 创新点：在线层次化 Spatial Cognitive Memory；统一 VLN 与 Object-Goal Navigation 的阶段接口；结构化、可解析的空间推理与控制约束。
- 为什么值得关注：该设计把 Agent memory 从文本摘要推进到可执行的空间状态，对具身 Agent 和 Computer Use 的状态管理都有启发。
- 局限与疑点：依赖 RGB-D、位姿、检测器和多次 VLM 调用；真实机器人测试集较小，开放复杂环境中的泛化仍待验证。
- 开源情况：论文提供项目页入口；截至核验时未确认完整代码与模型权重已公开。
- 原始链接：https://arxiv.org/abs/2606.08992

### A Multi-modal Agentic Co-pilot for Evidence Grounded Computational Pathology
- 类型：论文
- 方向：两者相关
- 发布日期：2026-06-06
- 来源与作者：arXiv；Zhe Xu、Zhengyu Zhang、Zhiyuan Cai、Jiahao Xu、Yijie Lin、Ziyi Liu、Junlin Hou、Hongyi Wang、Yuxiang Nie、Ling Liang、Yihui Wang、Yingxue Xu、Ronald Cheong Kin Chan、Li Liang、Hao Chen
- 一句话结论：PathPocket 将病理图像推理与分级医学证据检索结合，强调诊断结论的可追溯性。
- 核心内容：系统整理约 110,472 份分级证据文档，构建含 455 万实体、710 万关系的多模态病理 hypergraph。多 Agent 流程负责输入理解、证据检索、过滤和诊断生成，支持文本、ROI 与 gigapixel WSI。作者称其在超过 20 万真实案例的多维评测中优于现有方法，并在用户研究中提升病理医生的准确率与信心。
- 创新点：把证据等级显式纳入多模态知识基础；以 hypergraph 支撑可追溯多 Agent 诊断；覆盖 ROI 到全切片图像的统一流程。
- 为什么值得关注：医疗 Agent 的价值不只在回答正确，还在于能否给出可审计证据链，PathPocket 对高风险垂直 Agent 有代表性。
- 局限与疑点：摘要未给出关键提升幅度；数据授权、跨机构泛化、证据更新和临床部署安全仍需完整论文与外部验证。
- 开源情况：截至核验时未确认代码、模型或完整数据公开。
- 原始链接：https://arxiv.org/abs/2606.08093

### Self-Evolving Scientific Agent Discovers Generalizable Physically-Reasoned Fluid Control
- 类型：论文
- 方向：Agent
- 发布日期：2026-06-07
- 来源与作者：arXiv；Boai Sun、Wenjin Guo、Zongmin Yu、Liu Yang
- 一句话结论：LLM Agent 可通过“仿真观测-诊断-代码改写”循环生成可读的物理控制器，而非只优化黑盒权重。
- 核心内容：该工作让科学 Agent 在流固耦合仿真中观察多模态证据、诊断动态行为并迭代修改控制源码。目标是控制欠驱动双关节仿生鱼到达空间目标。作者报告所得控制器无需重训练或目标特定分支，即可泛化至未见静态目标与曲线追踪，并保留完整演化日志。
- 创新点：以源代码演化替代纯参数优化；把物理证据、可解释控制结构和自动实验闭环结合；提供可审计的 discovery trace。
- 为什么值得关注：这是 Agentic Science 从“辅助写代码”走向“提出、验证并修正可解释机制”的具体案例。
- 局限与疑点：验证集中于单一仿真系统；泛化是否来自通用科学推理、提示设计或环境先验尚难区分，真实物理部署未验证。
- 开源情况：截至核验时未确认代码和仿真环境已开放。
- 原始链接：https://arxiv.org/abs/2606.08405

## 今日趋势观察
近期工作共同把“记忆”从长文本缓存转向结构化状态：H2HMem 强调跨人、跨会话、跨模态信息，SpaceVLN 则把记忆具体化为空间图与地标证据。具身方向开始正面处理高层推理与低层控制互相干扰的问题，并借助示范检索约束在线 RL。垂直 Agent 也在加强证据可追溯性与流程分工，而不是只追求端到端回答。值得持续跟踪的方向是 multimodal memory、VLA 的稳定 RL、可验证的闭环规划，以及科学和医疗场景中的审计机制。
