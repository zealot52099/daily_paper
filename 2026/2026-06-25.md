### OpenThoughts-Agent: Data Recipes for Agentic Models
- 类型：论文
- 方向：Agent
- 发布日期：2026-06-23
- 来源与作者：arXiv；Negin Raoof、Richard Zhuang、Marianna Nezhurina、Etash Guha、Ludwig Schmidt、Benjamin Feuer 等
- 一句话结论：论文给出全开放的 agentic model 训练数据配方，并用 100+ 消融实验说明“任务来源与多样性”是泛化能力关键。
- 核心内容：OpenThoughts-Agent 针对开放社区缺少通用 Agent 训练数据配方的问题，构建了数据策划流水线。作者组装 100K 训练样本微调 Qwen3-32B，在 7 个 agentic benchmarks 上达到 44.8% 平均准确率，比 Nemotron-Terminal-32B 的 40.9% 高 3.9 个百分点。论文结论是，该数据集在 compute-controlled 比较中随训练规模有稳定收益。
- 创新点：全开放数据流水线；100+ 控制消融；跨 benchmark 的 agentic 数据 scaling 分析。
- 为什么值得关注：Agent 训练正在从单 benchmark 技巧转向数据工程和可复现配方，这篇对开源 Agent 模型训练很实用。
- 局限与疑点：结果仍围绕 7 个 benchmark；真实长时任务、工具生态变化和安全约束下的泛化仍待验证。
- 开源情况：训练集、流水线、实验数据和模型计划/已在 https://openthoughts.ai 发布。
- 原始链接：https://arxiv.org/abs/2606.24855

### GUI vs. CLI: Execution Bottlenecks in Screen-Only and Skill-Mediated Computer-Use Agents
- 类型：论文
- 方向：Agent
- 发布日期：2026-06-22
- 来源与作者：arXiv；Xiao Zhou、Siyue Zhang、Yilun Zhao、Arman Cohan、Chen Zhao 等
- 一句话结论：在匹配任务和验证器后，GUI 与 CLI Agent 的瓶颈并不相同：GUI 难在长链视觉交互，CLI 难在技能覆盖。
- 核心内容：论文构建 440 个桌面任务，覆盖 18 个应用和 12 类工作流，让 screen-only GUI agents 与 skill-mediated CLI agents 在相同目标、状态和最终验证器下比较。最强 GUI agent full pass rate 为 59.1%，原始技能 CLI 为 48.2%；但加入 verifier-guided skill augmentation 后 CLI 升至 69.3%。论文结论是，CLI 劣势很大部分来自技能接口不完整，而非模型本身。
- 创新点：匹配执行层 benchmark；区分 modality-native action；用验证器引导技能增强量化 CLI 上限。
- 为什么值得关注：Computer Use 产品设计需要决定“看屏幕点按钮”还是“生成脚本/技能”，这篇给了较清晰的实验拆解。
- 局限与疑点：增强技能可能引入额外工程成本；桌面任务是否代表企业复杂流程仍需观察。
- 开源情况：论文页未明确给出代码链接。
- 原始链接：https://arxiv.org/abs/2606.24551

### EG-VQA: Benchmarking Verifiable Video Question Answering with Grounded Temporal Evidence
- 类型：论文
- 方向：多模态
- 发布日期：2026-06-23
- 来源与作者：arXiv；Linpeng Huang、Weixing Chen、Zexin Chen、Yang Liu、Liang Lin
- 一句话结论：VideoQA 不能只看答案对不对，还必须验证模型是否定位到正确时间证据。
- 核心内容：EG-VQA 提出带 temporal evidence 标注的开放式 VideoQA 评测，包含 2,067 个视频和 11,838 个 QA。作者提出 EG-F1，同时衡量时间对齐和语义一致性。实验发现强闭源模型也会出现答案正确但证据定位不可靠的问题；EG-Reasoner 用显式证据监督后，在开源模型中达到 SOTA，并在反事实等推理任务上提升明显。
- 创新点：证据可验证的 VideoQA 协议；EG-F1 指标；显式 evidence-grounded reasoning 训练。
- 为什么值得关注：视频理解正进入“可解释、可追溯”阶段，对长视频 Agent、安防、教育和工业质检都有意义。
- 局限与疑点：数据规模中等；证据标注成本高，跨领域泛化和长视频密集事件仍待验证。
- 开源情况：论文页未明确给出代码/数据链接。
- 原始链接：https://arxiv.org/abs/2606.24797

### SAFARI: Scaling Long Horizon Agentic Fault Attribution via Active Investigation
- 类型：论文
- 方向：Agent
- 发布日期：2026-06-23
- 来源与作者：arXiv / ICML 2026 AIWILD Workshop；Chenyang Zhu、Jiayu Yao、Kushal Chawla、Sambit Sahu、Erin Babinsky 等
- 一句话结论：长轨迹 Agent 调试不应把全量 trace 塞进上下文，而应通过工具化主动调查定位故障。
- 核心内容：SAFARI 面向多步、多 Agent 任务的失败归因，使用可搜索/读取轨迹片段的工具箱和 Short-Term Memory 替代线性上下文加载。论文报告在 Who&When 数据集 1M token 预算下比 SOTA 高 20%，在 TRAIL GAIA subset 的 25K token 预算下高 19%。当目标故障位于模型原生上下文 5 倍之外，SAFARI 仍保持 0.58 precision。
- 创新点：主动调查式故障归因；轨迹工具箱 + STM；超上下文窗口故障定位评测。
- 为什么值得关注：真实 Agent 系统的难点正在转向可观测性、调试和责任归因，这篇很贴近生产问题。
- 局限与疑点：依赖轨迹结构化和工具可用性；归因准确性是否能转化为自动修复能力尚未证明。
- 开源情况：论文页未明确给出代码链接。
- 原始链接：https://arxiv.org/abs/2606.24626

### video-SALMONN-R^3: Learning to ReWatch, ReAsk, and ReAnswer for Efficient Video Understanding
- 类型：论文
- 方向：多模态
- 发布日期：2026-06-23
- 来源与作者：arXiv；Yixuan Li、Guangzhi Sun、Yudong Yang、Wei Li、Zejun Ma、Chao Zhang
- 一句话结论：视频大模型可先粗看定位关键片段，再以更高时空精度复看，并通过 RL 端到端学习该策略。
- 核心内容：论文针对视频 LLM 因算力和显存限制只能低帧率/低分辨率采样而遗漏关键信息的问题，提出 ReWatch、ReAsk、ReAnswer 三机制。模型先给直接答案，再复看局部片段修正；复看时重新注入问题以提升 query adherence。作者称该方法无需 CoT cold-start 和昂贵 CoT 标注，优于 base model 和 QA-SFT baseline，且计算成本低于既有 re-watch 方法。
- 创新点：无 CoT 冷启动的 RL re-watch；re-answer 缓解答案先验；re-ask 强化局部复看与问题对齐。
- 为什么值得关注：长视频理解的关键是自适应计算，而不是盲目增加帧数；该方向适合视频 Agent 和多模态检索。
- 局限与疑点：代码、模型、数据仅承诺接收后发布；真实超长视频和音频强相关任务仍需更多结果。
- 开源情况：代码、模型、数据将于接收后公开。
- 原始链接：https://arxiv.org/abs/2606.24477

### PHANTOM: A Large-Scale Dataset of Multimodal Adversarial Attacks for Vision-Language Models
- 类型：论文 / 数据集
- 方向：多模态
- 发布日期：2026-06-23
- 来源与作者：arXiv；Simone Gallivanone、Hossein Khodadadi、Mauro Dore、Mauro Medda、Nicola Franco
- 一句话结论：PHANTOM 提供 47,524 个 VLM 对抗样本，降低多模态安全评测与防御研究门槛。
- 核心内容：数据集覆盖 10 个高层类别、55 个 harmful intent 子类，整合并扩展多个已有 benchmark，总计 7,826 个 intents。作者强调预生成攻击样本可减少研究者生成攻击的计算成本和复杂度。论文结论是，该资源可用于评测 VLM 鲁棒性、微调攻击生成模型、压力测试防护栏。
- 创新点：大规模预生成多模态攻击集；细粒度 harmful intent 分类；整合多来源 benchmark。
- 为什么值得关注：VLM 安全评测仍缺统一、可复现实验资源，PHANTOM 对红队与防御都很有价值。
- 局限与疑点：预生成攻击可能很快被新模型/新防御过拟合；真实交互式 jailbreak 仍更复杂。
- 开源情况：数据集已发布在 Hugging Face：https://huggingface.co
- 原始链接：https://arxiv.org/abs/2606.24388

### Agentic Collaborative Cognition for Zero-Shot 3D Understanding
- 类型：论文
- 方向：两者相关
- 发布日期：2026-06-23
- 来源与作者：arXiv / ECCV 2026；Wenxin Wang、Bo Zhang、Feng Chen、Zixuan Wang、Wen Li、Changsheng Li、Yinjie Lei
- 一句话结论：把 3D 理解拆成规划视角与感知建图两个 Agent，可以缓解视频关键帧视角有限的问题。
- 核心内容：论文提出 collaborative multi-agent framework：Planning Agent 负责高层视角规划并补充关键视角，Perception Agent 维护 object-level attributes 和跨视角一致实例 ID，形成 holistic cognitive map。两个 Agent 闭环迭代，直到信息足够完成任务。作者报告在 6 个 benchmark 上达到 SOTA，包括 ScanRefer Acc@0.5 提升 11.1%、3D-assisted dialog BLEU-1 提升 14.6、SQA3D EM 提升 2.1。
- 创新点：规划/感知分工的多 Agent 3D 理解；结构化 cognitive map；跨视角实例一致性维护。
- 为什么值得关注：这是多模态 Agent 与 3D 场景理解的结合，适合机器人、室内导航和空间问答持续跟踪。
- 局限与疑点：依赖候选视角生成和 MLLM 视觉能力；实际机器人在线感知中的时延、遮挡和误差累积未充分验证。
- 开源情况：项目页：https://zhangbo135.github.io
- 原始链接：https://arxiv.org/abs/2606.24649

## 今日趋势观察
今天的高价值内容集中在两个方向：一是 Agent 从“能做任务”转向“可训练、可评测、可调试、可治理”，代表是 OpenThoughts-Agent、SAFARI、GUI vs. CLI 和共享/自治评测类工作。二是多模态从单点准确率转向 evidence grounding、结构一致性和安全鲁棒性，VideoQA、视频复看和 PHANTOM 都体现了这一点。值得持续跟踪的方向包括：Agent 数据配方与 RL 训练、Computer Use 执行层设计、长轨迹故障归因、视频证据定位、VLM 对抗安全，以及 3D/视频中的结构化世界建模。

## 今日必读
1. OpenThoughts-Agent：最值得读，原因是它直接触及开源 Agent 模型训练的数据配方问题，并释放数据、流水线和模型，对复现实验和二次训练价值最高。
2. GUI vs. CLI：适合所有做 Computer Use/桌面 Agent 的团队阅读，因为它把 GUI 与 CLI 的能力差异拆成了可操作的工程瓶颈，而不是泛泛比较模型强弱。
