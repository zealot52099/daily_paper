### Native Active Perception as Reasoning for Omni-Modal Understanding
- 类型：论文
- 方向：两者相关
- 发布日期：2026-06-17
- 来源与作者：arXiv；Zhenghao Xing、Ruiyang Xu、Yuxuan Wang、Jinzheng He 等
- 一句话结论：OmniAgent 把长视频理解从“看完整段”改成按需感知的 Observation-Thought-Action 循环，显著降低长上下文负担。
- 核心内容：论文提出 POMDP 式 omni-modal agent，用动作选择性提取音视频线索并写入持久文本记忆。训练上结合 Agentic SFT 与 TAURA 强化学习，把 credit assignment 聚焦到关键发现轮次。摘要报告其在 10 个视频/音视频理解基准上达到开源模型 SOTA，7B 模型在 LVBench 上超过 Qwen2.5-VL-72B（50.5% vs. 47.3%）。
- 创新点：原生活跃感知循环；turn-aware RL 奖励重标定；长视频推理复杂度与原始时长解耦。
- 为什么值得关注：这是多模态 Agent 与长视频理解交叉中很强的系统化方案，直接回应“看全部帧”不可扩展的问题。
- 局限与疑点：代码链接在 arXiv 页面未展开，需复核；收益可能依赖动作空间设计与视频任务可分解性。
- 开源情况：arXiv comments 声明 Code and models，但未显示具体 URL。
- 原始链接：https://arxiv.org/abs/2606.19341

### Beyond the Current Observation: Evaluating Multimodal Large Language Models in Controllable Non-Markov Games
- 类型：论文
- 方向：两者相关
- 发布日期：2026-06-17
- 来源与作者：arXiv；Shengyuan Ding、Xilin Wei、Xinyu Fang、Haodong Duan、Dahua Lin 等
- 一句话结论：RNG-Bench 发现当前 MLLM 在闭环任务中的主要残差错误来自遗忘历史观察，而不只是动作选择差。
- 核心内容：论文构建 Reconstructive Non-Markov Games，包括 Matching Pairs 与 3D Maze，专门隔离“不可见历史状态重构”能力。基准控制网格大小、视觉模式与观察模态，并引入 head-to-head duel 与 Memory Gap 指标。最难配置需要约 128K tokens 与 350 张图像输入，前沿 MLLM 仍远未饱和。
- 创新点：把多模态记忆从一般推理中剥离；Memory Gap 区分遗忘与决策失误；可控非马尔可夫交互环境。
- 为什么值得关注：Agent 走向真实闭环任务时，历史视觉记忆会成为核心瓶颈，这个基准比静态 VQA 更贴近部署问题。
- 局限与疑点：任务仍是游戏化环境，能否代表开放世界任务需进一步验证；未见代码/数据链接。
- 开源情况：arXiv 页面未声明开放代码、模型或数据。
- 原始链接：https://arxiv.org/abs/2606.19338

### RODS: Reward-Driven Online Data Synthesis for Multi-Turn Tool-Use Agents
- 类型：论文
- 方向：Agent
- 发布日期：2026-06-17
- 来源与作者：arXiv；Ruishan Fang、Siyuan Lu、Chenyi Zhuang、Tao Lin
- 一句话结论：多轮工具调用 RL 的关键不是更多静态样本，而是持续生成位于能力边界的高方差样本。
- 核心内容：论文指出 GRPO 梯度主要来自 rollout reward variance 最高的任务，即成功/失败接近平衡的能力边界样本。RODS 用训练中已有 rollout 的奖励方差作为零额外推理成本的边界检测器，再合成 API 拓扑和依赖深度相近的新任务。摘要报告从 400 个人工种子、约 800 活跃样本出发，可接近 17K 离线样本流水线效果，并少用约 20 倍轨迹。
- 创新点：用 reward variance 做在线数据选择；动态 replay buffer 与策略共同演化；面向多轮工具结构的 resampling。
- 为什么值得关注：Agent RL 正从“有环境就训”转向数据效率问题，RODS 提供了可操作的数据闭环。
- 局限与疑点：摘要称 controlled setting，泛化到真实 API 生态和长尾失败模式仍需验证。
- 开源情况：arXiv 页面未声明开放。
- 原始链接：https://arxiv.org/abs/2606.19047

### TRAP: Benchmark for Task-completion and Resistance to Active Privacy-extraction
- 类型：论文
- 方向：Agent
- 发布日期：2026-06-17
- 来源与作者：arXiv；Moon Ye-Bin、Nam Hyeon-Woo、Baek Seong-Eun、Yejin Yeo、Tae-Hyun Oh
- 一句话结论：能正确使用隐私字段完成任务的 Agent，也更容易被诱导泄露这些字段，prompt 防御存在结构性上限。
- 核心内容：TRAP 将每个场景拆成含私密字段的文档、需要用字段调用工具的任务查询，以及诱导泄露的攻击查询。论文评估 22 个模型，发现所有模型族都有非零泄露，且 instruction-following 能力与泄露率相关。作者进一步给出 softmax 模型下软约束防御无法同时保证高成功率与零泄露概率的论证，并提出 structural private field isolation。
- 创新点：同时评测任务完成与主动隐私抽取抵抗；把隐私字段隔离为哈希键；给出 prompt 防御上限分析。
- 为什么值得关注：它直接击中企业 Agent 落地中的“既要用敏感信息，又不能说出敏感信息”矛盾。
- 局限与疑点：结构隔离会增加系统工程复杂度；对跨工具链、多步间接泄露的覆盖仍需看完整数据集。
- 开源情况：arXiv 页面未声明开放。
- 原始链接：https://arxiv.org/abs/2606.18996

### Visual-OPSD: Cross-Modal On-Policy Self-Distillation for Efficient Unified Multimodal Reasoning
- 类型：论文
- 方向：多模态
- 发布日期：2026-06-17
- 来源与作者：arXiv；Pengyu Li、Zhitao Gao、Lingling Zhang、Muye Huang 等
- 一句话结论：多步生成“visual thoughts”的成本可通过 privileged trace 蒸馏压缩到纯文本推理中。
- 核心内容：论文观察到 Unified Multimodal Models 生成 visual thoughts 的像素本身贡献有限，但生成路径携带有用语义。Visual-OPSD 让 teacher 看到 privileged VT trace，student 只看问题，并用 on-policy trajectory 上的 token-level JSD 蒸馏。摘要报告在 9 个基准上较生成式 teacher 提升 3.40pp，同时达到 14.3 倍加速。
- 创新点：跨模态 OPSD；用 KL gap 诊断生成路径语义；将视觉中间过程压缩进文本推理。
- 为什么值得关注：如果结果稳健，它给多模态推理降本提供了比直接删掉视觉生成更有原则的路线。
- 局限与疑点：依赖 privileged VT trace；是否能迁移到开放视频、机器人和真实空间任务还不清楚。
- 开源情况：arXiv 页面未声明开放。
- 原始链接：https://arxiv.org/abs/2606.18974

### Does VLA Even Know the Basics? Measuring Commonsense and World Knowledge Retention in Vision-Language-Action Models
- 类型：论文
- 方向：两者相关
- 发布日期：2026-06-17
- 来源与作者：arXiv；Nikita Kachaev、Andrey Moskalenko、Matvey Skripkin 等
- 一句话结论：Act2Answer 用动作回答知识题，揭示 VLA 微调后会在复杂语义知识上明显落后于源 VLM。
- 核心内容：论文把 VLM 知识基准改写成短桌面 episode，模型通过一次物体放置动作选择答案，从而减少低层控制混淆。作者评估 7 个 VLA 与 9 个 VLM baseline，并用 layerwise intent probing 定位知识信号。摘要显示 VLA 对简单概念尚可，但丰富语义类别落差更大，VQA co-training 与更好的知识保持相关。
- 创新点：action-grounded knowledge evaluation；层级 intent probing；区分知识遗忘与控制失败。
- 为什么值得关注：VLA/机器人 Agent 热度很高，但“会动”不等于“保留常识”，该工作提供了诊断工具。
- 局限与疑点：桌面单步动作仍简化了真实机器人任务；项目页 URL 在 arXiv 页面未展开。
- 开源情况：arXiv comments 声明 Project page，但未显示具体 URL。
- 原始链接：https://arxiv.org/abs/2606.19297

## 今日趋势观察
今天的高价值论文共同指向一个趋势：多模态模型正在从静态问答转向闭环、长时程、带记忆的 Agent 形态。长视频理解、VLA、工具调用 RL 都不再只拼 backbone，而是在感知调度、历史记忆、训练样本选择和隐私隔离上做系统设计。值得持续跟踪的方向包括：主动感知式视频 Agent、非马尔可夫多模态记忆基准、工具调用 RL 的在线数据合成，以及企业 Agent 的结构化隐私防护。

## 今日必读
首推 OmniAgent，因为它把多模态理解、Agent 动作循环和测试时扩展性结合得最完整，且已被 ICML 2026 接收。其次推荐 TRAP，它虽然不是性能榜论文，但对真实 Agent 产品的安全架构约束很强，尤其是“prompt 防御无法同时高任务成功与零泄露”的结论值得工程团队认真消化。
